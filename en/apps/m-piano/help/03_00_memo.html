Memo

Chord Progression Memo

What is Memo?

Memo is a feature for saving and organizing chord progressions.
You can store chords found in Forward or Reverse as memos, and play or share them.

Screen Layout

The main elements of the Memo screen are as follows.

① Page Title
The title of the currently displayed memo page. You can edit the title directly to rename the page.

② Page Selection Button
Displays a list of registered memo pages. Selecting another page will display the memos on that page.

③ Add Page Button
Creates a new memo page. (Pro version)

④ Import Button
Imports chord chart text and converts it into a chord progression.

⑤ Position Keyboard
Displays the chord fingering (position) on the keyboard.

⑥ Tap Button
Tapping a chord name button plays the chord and shows the corresponding keyboard position.

⑦ Transpose Button
Transposes the chord progression in semitone steps.

⑧ Email (Share) Button
Send the memo content via email or the share function. (Pro version)

⑨ Clear Button
Deletes all memos on the current page.

⑩ Edit Button
Allows editing such as reordering or deleting memos.

Full Screen Display

The memo chord buttons can be displayed in full screen mode.

This allows the chords to be shown in large size during live performance or practice.

Number of Memos

The number of memos that can be saved depends on the app version.

  • Free version : Limited
  • Pro version : Virtually unlimited

Page Management (Pro)

In the Pro version, memos can be managed by pages.

Each page can have its own title.

Examples

  • Practice
  • Live
  • Composition
  • Band Songs

Using pages makes it easier to organize chord progressions.

.mpmemo File

The .mpmemo file is the memo data format used by M-Piano.

When you receive a .mpmemo file by email,

Tap → M-Piano

Then

  • M-Piano launches
  • The memo is imported

This makes it easy to share chord progressions with friends or band members.
