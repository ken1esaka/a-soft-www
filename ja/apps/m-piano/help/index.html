M-Piano ヘルプ

ピアノコード検索・コード進行作成・メモ管理アプリ

M-Piano は
ピアノコード検索 / コード進行作成 / コードメモ管理 / コード譜インポート
を行うことができる、実践型のピアノコードツールです。

まず最初に（重要）

  • コードの再生は「鍵盤をタップ」して行います（スピーカーではありません）。
  • スピーカーアイコンは「ミュート（消音）」の切り替えです。
  • PRO 表示は Pro版で使える機能 です（購入前に見ても誤解しないよう、ヘルプ内で明確に区別します）。

タブの説明

Forward

コード名からピアノコードを検索します。
・コードの構成音
・鍵盤ポジション
・複数のボイシング
を確認できます。

Reverse

押さえた鍵盤からコード名を逆引きします。
作曲やコード分析に便利です。
Reverse からメモへ追加することもできます。

Memo

コード進行を保存・整理します。
・コード進行メモ
・コード譜インポート
・トランスポーズ（移調）
・メール共有（.mpmemo）
・全画面表示（メモ／タップボタン）

PRO Pro版ではページ管理が可能
メモできる数は理論上無制限（1ページのコード数もPro版では実質無制限）。

Setting

アプリの設定を行います。
・コード表示範囲（表示できるタイプの範囲など）
・表記設定（♯/♭ など）
・アプリ情報 / サポート

M-Pianoの主な機能

  • ピアノコード検索（Forward）
  • コード逆引き検索（Reverse）
  • コード進行メモ（Memo）
  • コード譜インポート（貼り付け解析）
  • メモのトランスポーズ（移調）
  • コード共有（メール）
  • .mpmemoファイルによるメモ共有（受け取ってダブルクリック→M-Piano起動→インポート）

Pro版で拡張される代表的な要素

  • PRO 表示できるコードタイプが増えます（表示範囲が拡張されます）。
  • PRO オンベース（/C など）が使えます。

オンベース（PRO）

PRO オンベース（例：C/E、D/F# など）は Pro版のみ利用できます。
オンベースを選ぶと、上部の大きな鍵盤に ポジションマーク が表示され、
再生音も ベース音を含んだ形 で確認できます。

コード譜インポート

コード譜を貼り付けると、コード進行を自動解析してメモに取り込みます。
詳細は インポートの説明 を参照してください。 

C        G
夜の風が　静かに吹いて

Am       F
街の灯り　揺れている

C        G
小さな歌を　口ずさめば

F        G        C
明日へ続く　道になる

Pro版について

Pro版では、学習・作曲・整理をより快適にするための機能が追加されます。
ヘルプ内では PRO マーク付きの項目が Pro版の内容です。

  • PRO メモのページ管理（タイトル別に複数ページで保存）
  • PRO メモできる数は理論上無制限（1ページのコード数も実質無制限）
  • PRO 表示できるコードタイプが増える（表示範囲が拡張）
  • PRO オンベース（/C〜/B）

Pro版の詳細は Pro版の説明 を参照してください。
